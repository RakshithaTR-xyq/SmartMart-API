# Retail Management System

## 📌 Project Overview
The Retail Management System is a backend application developed using Node.js, Express.js, and MySQL. It provides REST APIs to manage various retail store operations such as products, customers, employees, suppliers, and orders. This project demonstrates CRUD operations, modular routing, and database connectivity.

---

## 🚀 Features

- Product Management
- Customer Management
- Employee Management
- Supplier Management
- Order Management
- RESTful API Endpoints
- MySQL Database Integration
- Modular Route Structure
- Error Handling
- JSON Response Support

## ⚙️ Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
node server.js
```

or

```bash
npm start
```
